# wechat
提供项目框架搭建
通过tcp通信使多个客户端与服务端直接进行通信,传送文件
在开发板(客户端)显示客户端连接后的头像,点击头像可以进行聊天
使用字库,并添加简单小键盘

后续功能之后添加
